# finalproject_avo_m25_555
Комплексная платформа, которая позволяет пользователям регистрироваться, управлять своим виртуальным портфелем фиатных и криптовалют, совершать сделки по покупке/продаже, а также отслеживать актуальные курсы в реальном времени. Система состоит из двух основных сервисов: Сервис Парсинга (Parser Service) и Основной Сервис (Core Service)
